# 《英雄傳奇 I》中文說明書(精訊 珍藏版 36)

> 這是 Sierra《Hero's Quest / Quest for Glory I》台灣中文版說明書的掃描收錄——**軟體世界(SOFT-WORLD 精訊)** 發行的「珍藏版 36」。收錄於本專案,是為了保存 1990 年代台灣遊戲中文化的歷史記憶;所有權利屬於原出版者與原作者。

## 出處與致謝
- **中文版發行**:軟體世界研究開發中心(SOFT-WORLD Research Centre,高雄)
- **文字編輯**:黃啟禎　**美工編輯**:姚秀娟
- **原作**:Sierra On-Line(設計 Lori Ann Cole、Corey Cole)
- 版權屬原權利人所有,此處僅為歷史保存與研究/懷舊用途收錄。

## 封底前言(當年精訊的介紹文)

> 史四爾堡急徵英雄一名,沒有待遇,還可能送命,但自認為英雄的你豈能坐視不管?
>
> 自從堡主的一對兒女失蹤之後,他就獨自傷心地關在城堡裏,無心問政。原本用來保衛地方的部隊也解散了,只剩下一個警長維持秩序。因此,盜賊開始橫行,掠奪過往的行旅。妖怪也開始作祟,就連和平的墓地在夜晚也是鬼影幢幢。但是,這卻是你得以揚名立萬,成為英雄的好環境。
>
> 英雄傳奇是 Sierra 公司第一個冒險類型的角色扮演遊戲,你可以扮演戰士、魔法師或妙賊,每種職業的人都有獨特的技能,面對同樣的問題,可能需要不同的方法解決。你必須巧妙地運用這些技能使自己成為一個大英雄。

> 譯名小考:當年精訊把 **Spielburg** 譯作「史四爾堡」(封底)/「史匹爾堡」(附錄),thief 譯作「妙賊」;本中文化專案採用「史畢柏格」與「盜賊」。
